⚙️ Media Link Scanner Settings

Configure your extension to connect to the Media Link Scanner API

📡 Getting Started

Make sure the Media Link Scanner API server is running on your machine. Start the server with: npm run api:start or npx ts-node src/api/server.ts

API Configuration

The URL of your Media Link Scanner API server. Default is http://localhost:3001/api/external-scan
If your API server requires authentication, enter your API key or bearer token here.

Extension Behavior

When enabled, a new tab with the scanner app will open automatically when you send a URL to scan.
Receive notifications when scans start, complete, or fail.
